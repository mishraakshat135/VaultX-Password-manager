# VaultX

A secure password manager built using React, Express, MongoDB Atlas and JWT Authentication.

## Features

- User Signup/Login
- JWT Authentication
- AES Password Encryption
- User-specific Password Storage
- Password Visibility Toggle
- Responsive Cybersecurity-themed UI

## Tech Stack

- React
- Tailwind CSS
- Express.js
- MongoDB Atlas
- JWT
- CryptoJS
- Render
- Vercel
